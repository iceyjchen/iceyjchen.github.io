数据挖掘-逻辑回归

logistic函数：
$$
f(x)=\frac{e^{x}}{1+e^{x}}
$$
用$p_i=P(y_i=1|x_{i1},x_{i2},…,x_{ip})$作为因变量，得到logistic回归模型：
$$
\begin{array}{c}
p_{i}=\frac{\exp \left(\alpha+\beta_{1} x_{i 1}+\beta_{2} x_{i 2}+\cdots+\beta_{p} x_{i p}\right)}{1+\exp \left(\alpha+\beta_{1} x_{i 1}+\beta_{2} x_{i 2}+\cdots+\beta_{p} x_{i p}\right)} \
\ln \frac{p_{i}}{1-p_{i}}=\alpha+\beta_{1} x_{i 1}+\beta_{2} x_{i 2}+\cdots+\beta_{p} x_{i p}
\end{array}
$$
特点：线性分类器

Logit（logisticprobability unit）变换：

令$Logit(p_i)=\frac{p_i}{1-p_i}$

有$Logit(p_i)=\alpha+\beta_{1} x_{i 1}+\beta_{2} x_{i 2}+\cdots+\beta_{p} x_{i p}+\varepsilon_i$

变换特点：

  • $Logit(p_i)=\frac{p_i}{1-p_i} \in (0,+\infty)：正类$
  • $Logit(p_i)=\frac{p_i}{1-p_i} \in (-\infty,0)：负类$

应用：优势比OR

流行病学衡量危险因素作用大小的比数比例指标
$$
𝑂𝑅_𝑗=\frac{𝑃_1/(1−𝑃_1)}{𝑃_0/(1−𝑃_0)} \
P_1和P_0分别表示在X_j取值为c_1及c_0时的发病概率，OR_j称作多变量调整后的优势比，表示扣除了其他自变量影响后危险因素的作用
$$
表示自变量变化以后，发病概率的变化情况

对比某一危险因素两个不同暴露水平 $x_j=c_1与x_j=c_0 $的发病情况（假定其它因素的水平相同），其优势比的自然对数为
$$
\begin{array}{l}
\ln O R_{j}=\ln \left[\frac{P_{1} /\left(1-P_{1}\right)}{P_{0} /\left(1-P_{0}\right)}\right]=\operatorname{logit} P_{1}-\operatorname{logit} P_{0} \
=\left(\beta_{0}+\beta_{j} c_{1}+\sum_{t \neq j}^{m} \beta_{t} X_{t}\right)-\left(\beta_{0}+\beta_{j} c_{0}+\sum_{t \neq j}^{m} \beta_{t} X_{t}\right) \
=\beta_{j}\left(c_{1}-c_{0}\right)
\end{array}
$$
即
$$
OR_j = exp[\beta_j(c_1-c_0)]，\
若 X_{j}=\left{\begin{array}{lll}1 & \text { 暴露 } \ 0 & \text { 非暴露 }，\end{array}\right. \
c_{1}-c_{0}=1,\
则有 O R_{j} =\exp \beta_{j}, \quad \beta_{j}\left{\begin{array}{l}
=0, O R_{j}=1 \text { 无作用 } \

0, O R_{j}>1 \text { 危险因子 } \
<0, O R_{j}<1 \text { 保护因子 }
\end{array}\right.
$$

说明收入对工作状态有正关系，收入越高，工作概率越高

逻辑回归应用

正则化

$w_1x_1+w_2x_2 +w_0 =0$对应于平面的一根直线

w在数值上越小越好，这样越能抵抗数据的扰动

误差函数
$$
E=\sum_{i=1}^{n}\left(y_{i}-\frac{1}{1+e^{-\left(w_{1} x_{i 1}+w_{2} x_{i 2}+w_{0}\right)}}\right)^{2}+\lambda L_{i} \
L_1 = \sum_{i=0}^{m}|w_i| \
L_2 = \sum_{i=0}^{m}w_i^2
$$
最小化E值使得求解的权值尽可能相对较小

数据挖掘中，

  • $L_1$倾向于使得w要么取1，要么取0稀疏编码(交集在坐标轴上)，适合降维
  • $L_2$倾向于使得w整体偏小岭回归

数值优化

各个维度的输入如果在数值上差异很大，那么会引起正确的w在各个维度上数值差异很大

找寻w的时候，对各个维度的调整基本上是按照同一个数量级来进行调整

逻辑回归的优化方法

数值归一化 正则化 梯度下降法 分类个数 数据选择
LogisticR egression WithLBF GS 需要均值归一化，算法融入方差归一化 支持L2 LBFGS （收敛快，考虑二阶导数） 支持多分类 加载所有数据 都参与训练
LogisticR egression WithSGD 不归一化，需要专门在外面进行归一化 支持L1， L2 SGD 不支持多分类 随机从训练集选取（支持 MiniBatch Fraction）
